
# Installation

The Save System is distributed as a Unity package. You can install it in one of two ways using the Unity Package Manager

### Option 1: Install from Git Repository

This is the recommended option as it makes it easier to pull updates and bug fixes.

1. In Unity, navigate to `Window > Package Management > Package Manager`.
2. Click the `+` icon and choose `Add package from Git URL...`.
3. Enter the repository HTTPS URL: [link](https://github.com/Gavin-McG/DialogueSystem)

### Option 2: Install from Local Zip

1. Download the package contents from the repo as a .zip file.
2. In Unity, navigate to `Window > Package Management > Package Manager`.
3. Click the `+` icon and choose `Install package from disk...`.
4. Select the .zip file.
