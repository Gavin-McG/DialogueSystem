
# Example Sample

This sample is meant as a basic example of how the dialogue system can be used. It demonstrates many of the features provided by the system, included Text dialogue, Choice dialogue, stalled Events, Redirects, and variables. This is a good reference for if you're confused on how to use or interact with any of the provided features.

## UI Implementation

This sample also provides a very basic Scene and UI setup which displays the Dialoghe Graph that comes with the Sample. This can be very useful for reference when creating your own frontend/UI, or you can modify this implementation to be your own.
